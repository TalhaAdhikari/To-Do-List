# To-Do-List
A minimal To do list with pinned tasks and a status tracking
This is a minimal To Do list made by using basic HTML and JavaScript.

## Features

1)Has a pinned task for Important tasks 
2)User can add tasks during runtime
3)Tasks can be checked as completed 

## How to run
1)Download all the files
2)Run the 'index.html' file using any web browser
